# Backend do DevRadar
API feita NodeJS e Express para encontrar desenvolvedores próximos.

Para testar a API:
---------------------- 

### Clone o repositório
`$ git clone https://github.com/Rodr1go/backend.git`

### Entre no diretório do projeto via terminal
`$ cd backend`

### Instale as dependências com:
`$ yarn` ou `$ npm install`

### Inicie a aplicação
`$ yarn dev` ou `$ npm dev`
